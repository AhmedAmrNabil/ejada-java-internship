# Ejada Java internship

this repo contains my journey through the Ejada Java internship program.
the goal is to learn daily and create a small project everyday to test out new java concepts
each day will be a new folder with the name DayX where X is the day number, and inside that folder will be a java file with the name Main.java which will contain the code for that day.

## Folder structure

```
DayX/
	*.java # the java files
	Justfile # a runner script that will compile and run the java files
devenv.nix # devenv setup to spin up servies later and install dependencies
```
